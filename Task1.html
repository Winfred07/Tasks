Html language

HTML language is a basic knowledge you need to know to create websites.It is quite simple and all the websites are using it.Advanced websites are using a few more languages together with HTML.

Editor

To write a HTML code it's not a good idea to use a simple notepad because it miss a lot of features like code highlighting or UTF encoding support.

PSPad

At the beginning webdesigners may use Notepad++ which is a really good editor.

HTML page

HTML document can be created easily, basically it is just a text document.

Tag

HTML consists from marks (so called tags).By itself it allows to give the elemnts on the page some meaning and this is it's main task.Firstly HTML was used for graphic styling of pages but the resultig websites were confusing so it has been limited just to web content.

Doctype

The very first tag in HTML document is doctype. It defines that the text file is a HTML document.

HTML

Whole document is placed inside the html tag.This tag is divided into 2 tags: head and body.
